1. 概述与核心功能

Meta分析文献筛选器 Pro 是一个纯浏览器端运行的文献初筛工具，专为系统综述与 Meta 分析设计。数据全部存储在本地浏览器，无需部署服务器。

核心功能一览

功能说明
多数据库直连检索PubMed、Europe PMC、Crossref、OpenAlex 一键检索
中文数据库辅助为 CNKI、万方、维普、SinoMed 自动生成检索式
自动评分筛选基于关键词规则对人群 / 动物 / 体外研究自动打分
PRISMA 2020 流程图自动生成可导出 SVG/PNG 的筛选流程图
去重合并跨数据库自动去重（基于 PMID / DOI / 标题+年份）
文件导入支持 RIS、BibTeX、CSV、NBIB 格式批量导入
多格式导出支持 JSON、CSV 导出，兼容 EndNote / Zotero / Excel
离线可用离线激活码模式，无需网络即可使用完整功能

2. 激活与认证

首次使用需要激活。提供三种方式：

A在线注册 + 激活码

点击「注册」选项卡，填写邮箱、密码和激活码。注册后检查邮箱确认链接，然后登录即可。激活码一码一人，由管理员分发。

💡 在线模式支持跨设备数据同步（需配置 Supabase）。

B离线激活

点击「无法联网？离线激活」链接，输入管理员提供的离线激活码。支持以下格式：

  • META-PRO-2024-FULL / META-PRO-2025-FULL
  • MAS-* 或 META-* 前缀的任意码

💡 离线激活完全在本地验证，无需网络，适合机构内网环境。

C试用模式

点击「先试用看看」，无需任何激活码。限制：每次检索最多返回 15 条 记录。

⚠️ 清除浏览器数据会导致激活状态丢失。建议定期导出数据备份。

4. 数据库检索

4.1 构建检索式（PICO 框架）

在工作区上方按字段填写检索词，每行一个词组，同组内多个词用 分号 ; 或逗号分隔：

字段含义示例
人群 (Population)研究对象diabetes; type 2 diabetes
暴露/干预 (Exposure)处理或暴露因素metformin; "glucophage"
结局 (Outcome)关注的结局指标mortality; HbA1c
研究设计 (Design)文献类型限制RCT; "randomized controlled"
年份范围发表年份2015 — 2025

检索模式三种：

  • 宽检索：仅 P+E 组合，结果多
  • 中检索：P+E+O，适度精确
  • 严格检索：P+E+O+D，最精确
💡 建议先用宽检索，再用筛选器自动评分 + 人工复核逐步缩小范围。

4.2 直连数据库

勾选要检索的数据库，点击「🔍 检索选中数据库」按钮：

数据库类型说明
PubMed直连使用 Entrez API，支持完整 MeSH 检索式
Europe PMC直连欧洲 PMC API，补充 PubMed 未收录文献
Crossref直连侧重论文元数据和 DOI 检索
OpenAlex直连开放学术图谱，覆盖面广

每次最多获取数可在右上角设置（默认 500，试用模式限 15）。

4.3 中文数据库辅助

CNKI、万方、维普、SinoMed 四个中文库不直连，但工具会：

  1. 根据 PICO 自动生成适配各库的检索式
  2. 提供一键跳转链接，复制检索式到对应网站执行
  3. 检索结果导出后可通过「文件导入」回到筛选器

勾选中文库后，下方会显示各自定制的检索式，点击「🔗 打开检索」直接跳转。

💡 中文词请在 CN 人群词 / CN 动物词 / CN 暴露词 三个专属字段填写，工具会区别处理中英文检索式。

5. 手动导入文献

除了在线检索，也支持从文件导入已有文献库：

格式来源操作
RISEndNote, Zotero, Mendeley, CNKI点击「📥 导入」→ 选择 .ris 文件
BibTeXLaTeX, Google Scholar点击「📥 导入」→ 选择 .bib 文件
NBIBPubMed 导出点击「📥 导入」→ 选择 .nbib 文件
CSVExcel / Google Sheets点击「📥 导入」→ 选择 .csv 文件
JSON本工具导出的备份点击「📥 导入」→ 选择 .json 文件

导入时自动去重：如果导入了已存在的文献（同 PMID / DOI / 标题+年份），会自动合并来源信息而非重复添加。

⚠️ CSV 导入时确保包含标题列（title），否则去重效果不佳。

6. 自动评分与筛选

6.1 研究模式

切换研究模式会触发自动重评分：

模式评分依据
人群研究 (Human)匹配人群词得正分，动物词得负分；匹配暴露/结局词加分
动物研究 (Animal)匹配动物词（rat / mouse / rabbit 等）得正分；匹配人群词得负分
体外研究 (In Vitro)匹配细胞系 / Western Blot / PCR / ELISA 等体外标记词得正分

6.2 决策标记

每条文献可标记决策状态：

标记颜色含义
建议纳入🟢自动评分高于入围阈值
建议排除🔴自动评分低于排除阈值
最终纳入🔵人工确认纳入（手动点击 ✅）
最终排除⚪人工确认排除（手动点击 ❌），需填写排除原因
待人工判断🟡评分在中间区间，需人工阅读决定

6.3 筛选器工具栏

可使用关键词包含/排除过滤、按来源筛选、按决策状态筛选、排序（评分 / 年份 / 标题）等快速定位目标文献。

7. PRISMA 2020 流程图

页面底部自动维护符合 PRISMA 2020 标准的筛选流程图：

  • 自动计数：识别 → 去重 → 筛选 → 检索 → 纳入，各环节数字自动更新
  • 按研究模式：切换人群 / 动物 / 体外，流程图数据跟随变化
  • 排除原因汇总：人工排除时填写的原因自动汇总显示
  • 导出：点击「📥 下载 SVG」或「📥 下载 PNG」导出矢量或位图，可直接用于论文投稿

手动补充项：

  • 未检索到：填入检索了但未获取到全文的篇数

8. 数据导出

格式操作用途
JSON「📤 导出 JSON」完整备份，含评分、决策、来源等所有字段；可重新导入恢复
CSV「📤 导出 CSV」Excel 兼容，字段：决策、评分、研究类型、来源、PMID、DOI、原因、标题、年份、期刊、作者、命中词、摘要

9. 高级配置

9.1 评分调参

在「⚙️ 高级配置」面板中可调整：

参数默认值作用
入围阈值 (IncCut)3评分 ≥ 此值 → 建议纳入
排除阈值 (ExcCut)-3评分 ≤ 此值 → 建议排除

协同加分1P+E 同时出现额外加分
排除罚分-2命中排除词时的罚分

9.2 筛选词自定义

可自定义「包含词」（命中加分）和「排除词」（命中扣分），每行一个。

9.3 备份与恢复

  1. 导出 JSON → 保存到安全位置
  2. 需要恢复时 → 导入该 JSON 文件
  3. 或在新设备上导入 JSON + 重新激活即可继续工作

10. 常见问题

Q: 数据存在哪里？安全吗？

全部数据存储在浏览器的 localStorage 中，不上传到任何服务器。清除浏览器数据会丢失，请定期导出备份。

Q: PubMed 检索失败怎么办？

可能原因：① 网络问题（PubMed API 需科学上网）；② 检索式语法错误；③ 请求频率过高被限流。建议等待 3-5 秒后重试，或减少每次获取数量。

Q: 中文数据库为什么不能直连？

CNKI / 万方 / 维普 / SinoMed 没有公开 API。工具会生成专用检索式，需手动到相应网站执行检索，再将结果导出为 RIS 文件导入本工具。

Q: 换电脑了怎么迁移数据？

  1. 旧电脑：导出 JSON 文件
  2. 新电脑：打开筛选器 → 导入该 JSON
  3. 激活码需在新设备上重新输入（一码一人，支持多设备）

Q: PRISMA 图不够精细怎么办？

导出 SVG 格式，在 Illustrator / Inkscape 中进一步编辑排版。数字不会说谎——各环节计数基于实际筛选数据。

Q: 支持多少条文献？

浏览器 localStorage 通常限制 5-10 MB，约可存储数千条完整记录。建议大数据集分批次处理。
