使用帮助

检索流程

  • 设置关键词（疾病/人群、暴露/基因、结局、研究设计），选择检索模式
  • 勾选可直连数据库（PubMed / Europe PMC / Crossref / OpenAlex），点击"联网获取"
  • 勾选中文数据库（CNKI / 万方 / 维普 / SinoMed），系统生成各平台专用检索式，复制后去对应网站检索并导出题录文件，再导入本工具
  • 也可手动导入 RIS / CSV / BibTeX / NBIB / PubMed XML 文件

评分与筛选

  • 系统根据关键词命中情况自动评分，分为"建议纳入""建议排除""待人工判断"三档
  • 支持人群研究与动物研究分开评分，可在"研究对象模式"中切换
  • 标题命中权重高于摘要；多组关键词协同命中有加分
  • 可在"高级评分设置"中自定义各类规则权重和阈值
  • 人工复核后点击"最终纳入"或"最终排除"锁定结论

数据存储

  • 所有数据存储在浏览器本地（localStorage），不会上传到任何服务器
  • 清除浏览器数据会导致记录丢失，请定期使用"备份 JSON"功能

1. 概述与核心功能

Meta分析文献筛选器 Pro 是一个纯浏览器端运行的文献初筛工具，专为系统综述与 Meta 分析设计。数据全部存储在本地浏览器，无需部署服务器。

核心功能一览

功能说明
多数据库直连检索PubMed、Europe PMC、Crossref、OpenAlex 一键检索
中文数据库辅助为 CNKI、万方、维普、SinoMed 自动生成检索式
自动评分筛选基于关键词规则对人群 / 动物 / 体外研究自动打分
PRISMA 2020 流程图自动生成可导出 SVG/PNG 的筛选流程图
去重合并跨数据库自动去重（PMID / DOI / 标题+年份）
文件导入支持 RIS、BibTeX、CSV、NBIB 格式批量导入
多格式导出支持 JSON、CSV 导出，兼容 EndNote / Zotero / Excel
离线可用离线激活码模式，无需网络即可使用完整功能

2. 激活与认证

首次使用需要激活。提供三种方式：

A在线注册 + 激活码

点击「注册」选项卡，填写邮箱、密码和激活码。注册后检查邮箱确认链接，然后登录即可。激活码一码一人，由管理员分发。

💡 在线模式支持跨设备数据同步。

B离线激活

点击「无法联网？离线激活」链接，输入管理员提供的离线激活码。META-PRO-* / MAS-* 格式均支持。

💡 离线激活完全在本地验证，无需网络，适合机构内网环境。

C试用模式

点击「先试用看看」，无需任何激活码。限制：每次检索最多返回 15 条 记录。

⚠️ 清除浏览器数据会导致激活状态丢失。建议定期导出数据备份。

4. 数据库检索

4.1 构建检索式（PICO 框架）

在工作区上方按字段填写检索词，每行一个词组，同组内多个词用 分号 ; 或逗号分隔：

字段含义示例
人群 (Population)研究对象diabetes; type 2 diabetes
暴露/干预 (Exposure)处理或暴露因素metformin; "glucophage"
结局 (Outcome)关注的结局指标mortality; HbA1c
研究设计 (Design)文献类型限制RCT; "randomized controlled"
年份范围发表年份2015 — 2025

检索模式三种：宽检索（P+E）、中检索（P+E+O）、严格检索（P+E+O+D）。建议先用宽检索，再用筛选器逐步缩小范围。

4.2 直连数据库

数据库类型说明
PubMed直连Entrez API，支持完整 MeSH 检索式
Europe PMC直连欧洲 PMC API，补充 PubMed 未收录文献
Crossref直连侧重论文元数据和 DOI 检索
OpenAlex直连开放学术图谱，覆盖面广

4.3 中文数据库辅助

CNKI、万方、维普、SinoMed 不直连，工具会：① 自动生成适配检索式；② 提供一键跳转链接；③ 结果可导出后通过「文件导入」回到筛选器。

6. 自动评分与筛选

6.1 研究模式

模式评分依据
人群研究 (Human)匹配人群词得正分，动物词得负分；匹配暴露/结局词加分
动物研究 (Animal)匹配动物词（rat / mouse / rabbit 等）得正分；匹配人群词得负分
体外研究 (In Vitro)匹配细胞系 / Western Blot / PCR / ELISA 等体外标记词得正分

6.2 决策标记

标记含义
🟢 建议纳入自动评分高于入围阈值
🔴 建议排除自动评分低于排除阈值
🔵 最终纳入人工确认纳入（点击 ✅）
⚪ 最终排除人工确认排除（点击 ❌），需填写排除原因
🟡 待人工判断评分在中间区间，需人工阅读决定

7. PRISMA 2020 流程图

页面底部自动维护符合 PRISMA 2020 标准的筛选流程图：

  • 自动计数：识别 → 去重 → 筛选 → 检索 → 纳入，各环节数字自动更新
  • 按研究模式：切换人群 / 动物 / 体外，流程图数据跟随变化
  • 排除原因汇总：人工排除时填写的原因自动汇总显示
  • 导出：点击下载 SVG 或 PNG，可直接用于论文投稿

10. 常见问题

Q: 数据存在哪里？安全吗？

全部数据存储在浏览器 localStorage 中，不上传任何服务器。清除浏览器数据会丢失，请定期导出备份。

Q: PubMed 检索失败怎么办？

可能原因：① 网络问题（PubMed API 需科学上网）；② 检索式语法错误；③ 请求频率过高被限流。建议等待后重试。

Q: 中文数据库为什么不能直连？

CNKI / 万方 / 维普 / SinoMed 没有公开 API。工具生成专用检索式，需手动到相应网站执行检索后导出 RIS 导入。

Q: 换电脑了怎么迁移数据？

  1. 旧电脑：导出 JSON 文件
  2. 新电脑：打开筛选器 → 导入该 JSON
  3. 激活码需在新设备上重新输入

Q: 支持多少条文献？

浏览器 localStorage 通常限制 5-10 MB，约可存储数千条完整记录。建议大数据集分批次处理。
